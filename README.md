# unified-asset-brain-portal
It is a self-contained, interactive single-page web application frontend built using Tailwind CSS for modern utility layout styling and FontAwesome v6.4.0 for industrial machinery iconography.
It features premium typography combining Inter and JetBrains Mono. Crucially, it incorporates the KaTeX library to natively parse and render mathematical LaTeX variables in the browser, handling real-time scientific parameters like discharge pressures (14.5 bar) or rotational speeds (2950 RPM)
Functional Navigation Modules
The system implements client-side application state management across several functional modules:
Operations Dashboard: Tracks industry workforce knowledge retention metrics, unplanned downtime reductions, and equipment-specific relationships.  

Copilot RAG Channel: Features a sandbox environment simulating hybrid vector space indexing (ChromaDB/Neo4j) with pre-seeded query macros for immediate testing.  

RCA Agent: Simulates multi-agent diagnostics via a "5-Whys" causal chain to generate dynamic Corrective Action Plans (CAPA) based on equipment symptoms.  

Safety Compliance Auditor: Semantically evaluates Standard Operating Procedures (SOPs) against statutory benchmarks (like OISD-STD-116), running constraint validation algorithms.  

Maintenance Hub: Automatically flags and outputs dynamic, safety-vetted electronic Permits to Work (PTW) adhering to strict Lockout-Tagout (LOTO) requirements.  

Topology Network & Blueprint Explorer: Generates vector SVGs representing asset neighborhoods and active blueprint layouts with glowing, interactive hotspot overlays.  

Data Ingestion Center: Contains a working CSV parsing engine that actively reads unstructured tables to populate or update the portal's dataset on the fly.
