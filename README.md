# Unified Asset & Operations Brain - Enterprise Portal

An interactive, high-fidelity industrial intelligence prototype designed to streamline cross-referenced plant maintenance, automate multi-agent diagnostics, and enforce real-time regulatory compliance using **GraphRAG** paradigms.

## 🚀 Features

*   **Operations Dashboard:** Real-time tracking of knowledge retention indices, fragmentation metrics, engineering specs, and historical failure events.
*   **Expert Field Copilot (GraphRAG):** Context-aware conversational AI sandbox backed by simulated vector spaces (ChromaDB) and graph networks (Neo4j).
*   **Multi-Agent RCA Engine:** Structured "5-Whys" diagnostic investigator that maps physical symptoms to actionable Corrective and Preventive Actions (CAPA).
*   **Procedural Safety Auditor:** An interactive constraint-solving sandbox that semantically audits Standard Operating Procedures (SOPs) against statutory frameworks.
*   **Automated Work Order Desk:** Live generation and electronic signing of vetted Permits to Work (PTW) enforcing strict Lockout-Tagout (LOTO) protocols.
*   **Dynamic Topology & Blueprint Canvas:** Responsive vector SVGs mapping adjacent plant instrumentation zones alongside interactive blueprint hotspots.
*   **Universal Ingestion Parser:** A functional layout parser stream allowing on-the-fly uploading and mapping of tabular CSV operational logs into localized memory.
